# Histórias

> O livro vivo da sua família.

App SaaS para preservação de memórias familiares com timeline cronológica, otimizado para usuários sêniors.

## Stack

- **Next.js 14** (App Router)
- **TypeScript strict**
- **Tailwind CSS 4** + Design System próprio
- **Prisma*+ + **Supabase** (PostgreSQL + Storage)
- **Auth.js v5** (Google OAuth + Magic Link)
- **Stripe** (assinaturas mensais)
- **TanStack Query** (data fetching)
- **Storybook 8** (design system documentado)
